# Edu-Predict: AI-Powered Educational Portal

A comprehensive three-portal educational platform with AI-driven insights for student performance, mental health prediction, career path guidance, and counselor support.

## 🎯 Project Overview

Edu-Predict is an intelligent educational management system designed to support students, parents, professors, and counselors with data-driven insights and personalized recommendations.

### Three Main Portals

1. **Student/Parent Portal**
   - Academic dashboard with marks, attendance, test scores
   - AI-powered mental health prediction with risk analysis
   - Career path recommendations with roadmaps
   - Skill gap analysis and resource recommendations
   - Professor finder (location-based within 15 minutes)
   - Appointment scheduling for counseling and mentorship

2. **Professor Portal**
   - Student management and performance analytics
   - Mark and attendance upload
   - Skill certification recommendations
   - Mental health insights and counselor suggestions
   - Mentorship tools for poor performers
   - AI-powered student insights access

3. **Counselor Portal**
   - Mental health dashboard for all students
   - Risk analysis and performance metrics
   - Appointment scheduling and management
   - Follow-up appointment recommendations
   - Early dropout detection

## 🛠️ Tech Stack

- **Frontend**: React with TypeScript + Vite
- **Backend**: Node.js with Express
- **Database**: PostgreSQL
- **AI/ML Engine**: Python (Flask) with scikit-learn, TensorFlow, Pandas
- **Authentication**: JWT + bcrypt
- **Real-time**: WebSocket (Socket.io)
- **Chatbot**: NLP with spaCy
